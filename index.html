⚙️ 云端同步设置

📋 Google Sheets 同步设置步骤：

  1. 打开 Google Sheets，创建新表格，命名为"塞尔夫记录"
  2. 点击 扩展程序 → Apps Script
  3. 删除默认代码，粘贴下方提供的脚本代码
  4. 重要：在代码第1行，将 YOUR_SHEET_URL_HERE 替换为你的 Google Sheets 表格 URL
  5. 点击 部署 → 新建部署，类型选择"Web应用"
  6. 执行身份选择"我"，有权访问的用户选择"任何人"
  7. 点击"部署"后复制生成的 Web App URL
  8. 将 Web App URL 粘贴到下方并保存
从 Apps Script 部署后获取的 Web App URL

💡 提示：首次使用建议先点击"同步云端"保存数据到 Google Sheets，之后在另一台设备上点击"从云端加载"获取数据。
